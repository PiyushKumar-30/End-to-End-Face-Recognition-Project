Face Recognition System

About This Project

This face recognition system uses a webcam and JavaScript to detect and recognize faces in real-time. It leverages the Face API for accurate face detection and recognition.

Instructions

  • Ensure your webcam is connected and accessible by your browser.
  • Allow camera access when prompted.
  • Position your face within the video frame for detection.
  • The system will identify and label recognized faces.

Note

Currently, this system can only recognize a few people due to data copyright issues.

  • Currently system only recognised only two person Ronaldo and Messi
  • if you want system to recognised more person you need to add their images in data
